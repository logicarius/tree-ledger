Problem Statement

Environmental sustainability efforts require accurate tracking and analysis of tree and environmental data. However, such data is often unorganized, inconsistent, and difficult to analyze for beginners.

The goal of TreeLedger is to build a simple and structured data analysis system that:

Stores environmental and tree-related datasets in an organized manner

Cleans and preprocesses raw data

Performs basic statistical analysis

Produces clear visual insights

This project focuses on building a strong foundation in data analysis using Python, Pandas, and NumPy. The emphasis is on data understanding, structure, and clarity, not on machine learning or prediction at this stage.
